# useCircle

Hook used to a basic circle.

<MyUsePolgon />

## Configuration

**_(CircleConfig | undefined)_** `useCircle` accepts following configurations:

| Property |   Type   | Default | Description           |
| :------- | :------: | :------ | :-------------------- |
| `radius` | `number` | `100`   | Radius of the circle. |

## Returns

<PrimitiveReturns configType="CircleConfig" />

## Example

```ts
import { useCircle } from "react-use-polygon";

const circle = useCircle();
// or
const biggerCircle = useCircle({ radius: 200 });
```
