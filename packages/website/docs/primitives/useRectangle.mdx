# useRectangle

Hook used to a rectangle.

It can be created with `width` and `height` configurations.

<SimpleUseRectangle />

## Configuration

**_(RectangleConfig | undefined)_** `useRectangle` accepts following configurations:

| Property |   Type   | Default | Description               |
| :------- | :------: | :-----: | :------------------------ |
| `width`  | `number` |  `100`  | Width of the reactangle.  |
| `height` | `number` |  `50`   | Height of the reactangle. |

<PrimitiveConfig />

:::note

`useReactangle` always follows `usePolygon` which drawing vertex in counter-clockwise manner.

:::

## Returns

<PrimitiveReturns configType="RectangleConfig" />

## Example

```tsx
import { useRectangle } from "react-use-polygon";

// A rectangle
const rectangle = useRectangle({ length: 100, length: 50 });
```
