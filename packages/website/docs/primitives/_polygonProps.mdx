| Property     |   Type   | Default | Description                                                              |
| :----------- | :------: | :-----: | :----------------------------------------------------------------------- |
| `sides`      | `number` |   `4`   | The number of sides.                                                     |
| `sideLength` | `number` |  `100`  | The length of each side _(optional)_.                                    |
| `size`       | `number` |  `100`  | The diameter of circumscribed circle that polygon drawn in _(optional)_. |

:::tip

When defining any regular polygon, you can specify either `sideLength` or `size`.

If you define both, `size` will always take precedence.

:::
