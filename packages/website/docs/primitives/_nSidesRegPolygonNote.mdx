:::note

<code>{props.hookName}</code> create a **regular** {props.shapeName} by using <code>usePolygon(\{ ...config, sides: {props.sides} \})</code> under the hood.

If you wished to create more complex {props.shapeName}, please try [<code>usePolygon(\{ vertices \})</code>](/docs/primitives/usePolygon#configuration) instead.

:::
