# useSegment

Hook used to create segment of a circle.

<SimpleUseSegment />

## Configuration

:::note

`useSegment` will draw a segment counter-clockwise, starting from the right center position, as `useArc`.

If you wish to create a segment facing to other side, please use `rotation`.

:::

**_(SegmentConfig | undefined)_** `useSegment` accepts following configurations:

| Property |   Type   | Default | Description                         |
| :------- | :------: | :-----: | :---------------------------------- |
| `radius` | `number` |  `100`  | Radius of outer arc of the segment. |
| `angle`  | `number` |  `60`   | Angle of the segment, in degrees.   |

<PrimitiveConfig />

## Returns

<PrimitiveReturns configType="SegmentConfig" />

## Example

```tsx
import { useSegment } from "react-use-polygon";

const segment = useSegment({ radius: 80, angle: 45, rotation: 30 });
```
