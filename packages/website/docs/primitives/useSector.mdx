# useSector

Hook used to create sector of a circle.

<SimpleUseSector />

## Configuration

:::note

`useSector` will draw a sector counter-clockwise, starting from the right centre position, as `useArc`.

If you wish to create a sector facing to other side, please use `rotate`.

:::

**_(SectorConfig | undefined)_** `useSector` accepts following configurations:

| Property |   Type   | Default | Description                        |
| :------- | :------: | :-----: | :--------------------------------- |
| `radius` | `number` |  `100`  | Radius of outer arc of the sector. |
| `angle`  | `number` |  `60`   | Angle of the sector, in degrees.   |

<PrimitiveConfig />

## Returns

<PrimitiveReturns configType="SectorConfig" />

## Example

```tsx
import { useSector } from "react-use-polygon";

const sector = useSector({ radius: 80, angle: 45, rotate: 30 });
```
